Discussing About The Scope Of VAR , LET And CONST:

VAR: var is a javascript variable used to declare variables. var is the short form of Variable. it is create with the var keyword and the assignment operator (=) is used to assign value to that variable.
For an example: var x = 5;
But var has some limitations. It is function scoped that means if a var is declared inside a function then it can be accessible only inside the function. But if it is declared outside of the function then it become global scoped. It can be re-declared.

LET: let is a javascript variable used to declare variables. It is a block scope variable. That means it is only accessible within the function , loop or any other block that it was declared. let was first introduced in ECMAScript 2015 which is also known as ES6. It provides more comfortable way to declare variable compared to var.
For an example: let x = 10;
It can not be re-declared. It is more convenient to use let then var.

Const: const is a javascript variable used to declare variables in block scope. Variable declared with const means it is constant that means the value of the declared variable can not be changed. It is also introduced in ECMAScript 2015. Variables which are declared using const are immutable which means the values can not be changed once they are declared.
For an example: const x = 15;
It also can not be re-declared. It also can not be accessible outside of function , loop or any kind of block.

Use Cases Of NULL And Undefined:

Null:

1. Indicate The Value Null: If there is no value for any variable one can declare the value null of the variable intentionally. Example: let array = null;

2. Clear Object Properties: After set some property to an object It can be cleared by using null.

As A Placeholder: It can be used as a placeholder when working with data structures or arrays to reserve a place for a value.

Undefined:

1. Variable Declaration Without Initialization: If variable is declared but no value is assigned thed it will give undefined.

2. Function Parameters: If a function called with more then one arguments but it is provided one parameters then it will show undefined.

3. Object Properties: If we want to console a property that is not available in the object then it will show undefined.

4. Return Value: If there is no return statement in the function then it will show undefined.

REST API:

REST stands for Representational State Transfer which is an architectural style that defines a set of constraints to be used for creating web services. API stands for Application Programming Interface. REST API is a way of accessing web services in a simple and flexible way without having any processing. REST technology is generally preferred to the more robust Simple Object Access Protocol technology because REST uses less bandwidth. Its used to fetch or give some information from a web service. All communication done via REST API uses only HTTP request.
